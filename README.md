A simple CRUD app built with Nest.js framework and MongoDB.

Before running the app ensure you have installed Node.js and MongoDB.

## Installation

```bash
$ npm install
```

## Build

```bash
$ npm run build
```

## Running the app

```bash
$ npm run start
```

## Tests

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e
```
